## Überblick

Die Helden kommen nach vierzehn Tagen auf See in Luminaris an. An Bord der ***Eisernen Woge*** erblicken sie zum ersten Mal die gewaltige Stadt.

> *"Luminaris ist eine hungrige Bestie. Sie frisst Gold und spuckt Stahl aus."* — Kapitän Orik Steinbart
