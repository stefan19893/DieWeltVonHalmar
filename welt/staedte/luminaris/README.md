## Überblick

Luminaris ist eine florierende Handelsstadt im Westen des Kaiserreichs [Mareno](../../regionen/mareno.md). Die Stadt liegt auf der Insel im Mare Gigante und ist seit vielen Generationen eine bedeutende Handels- und Militärmacht.

> *"Luminaris ist eine hungrige Bestie. Sie frisst Gold und spuckt Stahl aus."*
> — Kapitän Orik Steinbart
